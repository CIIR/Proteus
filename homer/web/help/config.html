Parameters

port
  • Optional
  • Type: Integer
  • Valid values: Any open port
  • Default: A random open port

indexType
  • Optional
  • Type: String
  • Valid values: ciir.proteus.tools.apps.Lucene or ciir.proteus.tools.apps.Galago
  • Default: ciir.proteus.tools.apps.Galago

indexedFields
  • Optional
  • Type: String
  • Valid values: Any HTML tags in the document. However these should only be the fields that are indexed, such as location, organization, or person
  • Default: none

defaultKind
  • Required
  • Type: String
  • Valid values: all, ia-pages, ia-books
  • Default: none

notes
  • Galago only
  • Optional
  • Type: Object
  • Valid values: n/a
  • Default: none

Note Specific Parameters

    noteIndex
    • Required
    • Type: String
    • Valid values: n/a
    • Default: none
    noteFields
    • Optional
    • Type: Array of String
    • Valid values: Any HTML tags in the document. However these should only be the fields that are indexed, such as location, organization, or person
    • Default: none

kinds
  • Required
  • Type: Object
  • Valid values: n/a
  • Default: none

Kinds Specific Parameters

    ia-books, ia-pages, or all
    • At least one is required
    • Type: String
    • Valid values: n/a
    • Default: none
    Parameters
      index
      • Required
      • Type: Array of String
      • Valid values: Path to a valid index
      • Default: none
      button
      • Required
      • Type: String
      • Valid values: Any text, this will display in the UI allowing the user to restrict the search to the indexes specified in the index parameter.
      • Default: none

content
  • Required
  • Type: Object
  • Valid values: n/a
  • Default: none

Content Specific Parameters

    dir
    • Required
    • Type: String
    • Valid values: A valid directory where all the HTML/JS/etc. is found.
    • Default: none, but "web" is a good choice.

userdb
  • This is where users, labels, notes, etc. are stored.
  • Requried
  • Type: Object
  • Valid values: n/a
  • Default: none

Userdb Specific Parameters

    path
    • While there are other parameters, this is the only one you should change. Use the reference values above for all others.
    • Required
    • Type: String
    • Valid values: A valid directory where the database will be stored. Each instance of Proteus should have it's own database.
    • Default: none

broadcast
  • Optional
  • Type: Object
  • Valid values: n/a
  • Default: none

Broadcast Specific Parameters

    url
    • Required
    • Type: String
    • Valid values: n/a
    • Default: none
    port
    • Required
    • Type: Integer
    • Valid values: Any open port
    • Default: none
    messages
    • Required
    • Type: Array of String
    • Valid values: search, results, add-note, upd-note, del-note, add-tag, del-tag, login, logout, register, view-res
    • Default: none

title
  • Optional
  • Type: String
  • Valid values: n/a
  • Default: none

queryType
  • Galago only
  • Optional
  • Type: String
  • Valid values: simple or complex
  • Default: simple
