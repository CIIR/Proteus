Settings

The settings can be accessed via gear icon on the top right of the main search page or directly via the settings.html page e.g.:
http://bulgroo.cs.umass.edu:1234/settings.html
Here you can adjust settings and add labels.

Result Options

There are several options that affect how the search results are displayed:

  • Number of entities : How many of each entity type (if available) to display for each document.
  • Auto show duplicates : When checked, duplicate documents will be visible. If unchecked, you have to click the "Show Duplicates" link under a document to see them.
  • Auto show notes : When checked, any notes for a document will be show in the results.
  • Show found with queries : When checked, if a user applied a label to the document, the query that was used to find this document will be shown.
  • Show single terms in query builder : When checked, single terms will be shown in the query builder.
  • Allow query builder : When checked, the query builder link will be displayed when appropriate.

Note: these options are per user.

Labels

This is where you can create or modify the labels that show under documents. You may need to refresh the main search page to see any changes. Currently, labels can not be deleted. Anyone with access to that instance of Proteus will see these labels.
