Query Syntax

Fields

Fields are term(s) that have been indexed separately from the document text, allowing them to be retrieved more efficiently. Usually these are entity types such as "person" or "location". You can see which fields are available in the "Entity Types" displayed on the main search page.

Galago

Proteus uses Galago's "simple query language", which is a subset of Lucene's query language.

Ordered Window Search

To perform an ordered window search, simply surround the query with double quotes. For example the query “James Allan” will behave (mostly) as you would expect in Google.

Field Search

To search within a field the syntax is: <field>:<term>. For example to search for Professor Allan you may type: person:allan or person:”James Allan”
