Bibliography

To download a bibliography of a corpus, click the "download" icon () next to the corpus name on the main search page.

The following information will be displayed when available:

  • Type : Book, page, or note.
  • Title
  • Archive ID : Internet Archive identifier.
  • Author(s)
  • Publication Date
  • Publisher
  • Language
  • OCLC number : Unique identifier for the book. See https://en.wikipedia.org/wiki/OCLC.
  • Hathi Trust ID : If this book exists in the HathiTrust Digital Library, this is their identifier for the book.
  • Page Number : If available, this is the actual page number of the document.
  • Archive URL : The link to this book/page on the Internet Archive.
  • Miscellaneous : If this is a note, the text of the note will be in this field.

There are two tabs at the top of the page. One gives the data in TSV format, the other is more "human readable". If your browser allows, the TSV version will be downloaded automatically.
